CAPM Formulas to Memorize

Quick reference covering schedule, cost, communications, procurement, and benefit ratios.

Schedule & PERT
  • PERT Expected Time (TE): (O + 4M + P) / 6
  • Standard Deviation (sigma): (P - O) / 6
  • Variance: [(P - O) / 6]^2

O = optimistic, M = most likely, P = pessimistic. Critical path is the longest path with zero or least float.

Earned Value (Cost)
Core Definitions
  • Planned Value (PV): Planned % complete * BAC
  • Earned Value (EV): Actual % complete * BAC
  • Actual Cost (AC): Sum of actual costs to date
Performance Metrics
  • Schedule Variance (SV): EV - PV
  • Cost Variance (CV): EV - AC
  • Schedule Performance Index (SPI): EV / PV
  • Cost Performance Index (CPI): EV / AC
Forecasting
  • EAC (CPI holds): BAC / CPI
  • EAC (one-time issue): AC + (BAC - EV)
  • EAC (cost and schedule): AC + (BAC - EV) / (CPI * SPI)
  • Estimate to Complete (ETC): EAC - AC
  • Variance at Completion (VAC): BAC - EAC
Communications & Procurement
Communications
  • Channels: N (N - 1) / 2
  • N = number of stakeholders.
Procurement
  • Point of Total Assumption (PTA): (Ceiling Price - Target Price) * Buyer Share Ratio + Target Cost
Benefit & Financial Ratios
  • Benefit-Cost Ratio (BCR): Benefit / Cost
  • Return on Investment (ROI): (Gain - Cost) / Cost
  • Net Present Value (NPV): Sum of discounted cash flows (focus on interpretation)

Positive NPV and ROI > 0 typically support project selection.

Index Interpretation & Examples
  • SPI ≥ 1.0: On/ ahead of schedule (delivering value as fast as planned).
  • SPI < 1.0: Behind schedule.
  • CPI ≥ 1.0: On/ under budget (earning ≥ RM1.00 of value per RM1.00 spent).
  • CPI < 1.0: Over budget (e.g., CPI = 0.80 → only RM0.80 of value earned per RM1.00 spent).
